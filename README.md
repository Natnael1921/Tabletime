
Table Time

TableTime Delivery is a simple full-stack food ordering system where users can browse restaurants, view menus, place orders, and simulate payments. Restaurant owners can manage menus and handle incoming orders.

Features

User & Owner login

Browse restaurants & menus

Place and manage orders

Owner dashboard for menu and order management

Payment simulation (balance, deposit, withdraw)


Tech Stack

Frontend: React (Vite)

Backend: Node.js, Express, MySQL
